# Organizational_MCP_Assistant
An agentic AI service which follows fully MCP system and orchestrates using lang-graph where a conversational agent for organization people conversation as well as casual conversation, an entity recognition based booking system and general agent for global knowledge chit-chat is included.

## Procedure:
```bash
python -m venv env_name
```
```bash
env_name\Scripts\activate.bat
```
```bash
pip install -r requirements.txt
```

## Dependency:
- tensorflow
- transformers
- spacy
- langgraph
- mcp
- torch
- nltk
